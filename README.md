# 👁️ Retinal Fundus Based Heart Disease Prediction

## 📌 Project Title
Heart Disease Prediction Using Retinal Fundus Images and Multimodal Deep Learning

---

## 🧠 Abstract
This project is a deep learning-based system for predicting heart disease risk using retinal fundus images along with clinical data. It uses CNN-based feature extraction and multimodal fusion techniques to combine image and structured data for improved prediction accuracy.

The system provides a non-invasive and automated approach for early cardiovascular risk detection.

---

## ⚙️ Technologies Used
- Python 🐍  
- TensorFlow / Keras 🧠  
- Deep Learning (CNN, EfficientNet)  
- FastAPI ⚡  
- HTML, CSS, JavaScript 🌐  
- OpenCV & NumPy  
- Grad-CAM (Explainable AI)  

---

## 🏗️ System Workflow
- Retinal image preprocessing  
- Clinical data processing (Age, BP, Cholesterol, BMI)  
- CNN feature extraction from fundus images  
- Dense neural network for clinical features  
- Feature fusion (multimodal learning)  
- Final classification (Low / High risk)  

---

## 🚀 Features
- Upload retinal fundus image  
- Enter clinical parameters  
- AI-based heart disease risk prediction  
- Confidence score output  
- Explainable AI using Grad-CAM  

---

## 📊 Model Architecture
- EfficientNet / CNN for image feature extraction  
- Dense Neural Network for clinical data  
- Feature-level fusion  
- Sigmoid activation for final classification
